You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply -- returns the product of two integers. Example: /multiply?num1=5&num2=7
  • /github -- returns the full_name, id, and owner for a repository. Example: /github?query=users/amehlhase316/repos
  • /power -- returns the power of a base to the exponent. Example: /power?base=4&exponent=3
  • /temperature_conversion -- returns temperature conversion between Fahrenheit and Celcius. Example: /temperature_conversion?temperature=68&unit=F
